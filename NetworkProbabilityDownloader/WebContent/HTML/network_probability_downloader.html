MIDB PROBABILISTIC ATLASES


The MIDB Precision Mapping Brain Atlas

Explore atlases derived from personalized brain topography

The MIDB Precision Mapping Brain Atlas (MPMA) was created to support investigators in leveraging individual brain topographies to assist in characterizing various aspects of brain function and to aid in honing methods in medical discovery.

The MIDB Atlas consists of two derived atlas types and four sub-atlases:

Topographic Atlases:
    1)  A Single Subject Topographic Atlas: Consisting of single subject precision network maps
    2)  A Group Probabilistic Atlas: Consisting of Network based probabilities and associated regional parcellations

Integration Zone Atlases:
    3)  A Single Subject Integration Zone Atlas: Consisting of single subject precision integration zone maps
    4)  A Group Integration Zone Atlas: Consisting of group averaged integration zones

The MPMA is a community resource and is designed to house precision brain atlases from multiple sources and using multiple methods. To include your own version of a Topographic or Integration Zone Atlas, please contact XXXXX.

Big Data Science Making an Impact: Why use the MIDB atlas?

  • Define probabilistic network locations
    • Investigators who are specifically interested in characterizing the behavior of a particular network can use these label files established in a
      large group to confidently measure which network is being activated in an independent sample.
  • Resolve issues with large-scale topographic variation in participants.
    • The use of probabilistic atlases more precisely measures individual variation that contributes to spatial heterogeneity of a network, enabling
      investigators to more reliably measure group brain-behavior relationships.
  • No resting state data is available.
    • The MIDB probabilistic atlases can serve as a stereotactic target for brain stimulation when collection of low motion resting state fMRI data
      is not possible



Want to use individual subject brain maps?

The dlabel.nii files provided are in CIFTI format (refer to humanconnectome.org) , and provided in fs_LR32k format (91282 greyordinates).

All individualized brain maps from the over 9500 subjects from the ABCD study are downloadable through the National Data Archive (NDA).

We have developed an NDA-BIDS upload tool that facilitates researchers to upload their own ABCD processed data using the Brain Imaging Data Structure (BIDS) standards after a brief approval process.

Details of this dataset are more fully described in Hermosillo et al (2021): A Population Atlas of Individualized Functional Neural Network Topology: an ABCD Resource.


What are the MIDB Probabilistic Brain Atlases?

Every brain is unique. The precise configuration of each neuron’s firing pattern is established through a combination of developmental factors, genetics, and each person’s lived experience. Neurons self-organize into networks in an effort to make sense of organism-environment interactions. Perhaps unsurprisingly, emerging evidence suggests that the precise topography of each person’s networks is fairly unique.

What constitutes a brain network?

"Functional connectivity" has come to refer to correlated patterns of brain activity between regions. Two regions are said to be “functionally connected” if the time series of their activity is highly correlated. The reasoning behind this definition is that brain areas with similar activity patterns are likely to be communicating and sharing information. This is likely true for regions that are highly anti-correlated, but this remains to be fully explored within the literature.

While the best approach to defining neural networks is still a subject of debate, we can loosely define them as a collection of brain regions (parcels, grayordinates, or voxels) that have shared patterns of co-activation, which in fMRI manifests as co-activation of the BOLD response between regions. This particular set of downloadable atlases were generated using a technique called template matching (see details below).

How is an individual’s brain network defined?

Template matching compares the whole-brain connectivity of each grayordinate or voxel for a given subject’s brain to the connectivity patterns of a set of template networks. The templates are generated from population averages and therefore provide a basic foundation for how we generally expect the networks to be structured. Networks for each subject were produced using a codebase that can be accessed on gitlab.   We’ve generated a probabilistic ROI set based on network similarity to networks defined by Gordon and colleagues. After mapping networks for each participant, the probability maps were generated using 2995 participants with 10 minutes of motion-censored data. Precision brain maps for this dataset were generated by processing through the DCAN processing pipeline, which is a modified version of the Human Connectome Project Processing pipeline (Glasser et al. 2013).

Read about the full details of the template matching procedure here: Hermosillo et al (2021) - Population Atlas of Individualized Functional Neural Network Topology: an ABCD Resource. .

How were the atlases generated?

The MIDB Probabilistic atlases are a collection of network atlases, each constructed by calculating the probability of observing a given network in thousands of individual brain maps. Multiple versions of individual-specific maps for each subject are available to download via the NIH that were generated using varying amounts of minutes, different community detection methods, and even concatenated rest and task data in order to optimize scan duration. Time series data were motion-censored to reduce the artifacts in the data induced by effects of head motion.

Scan details: For the functional scans, each subject underwent twenty minutes of resting-state data collection, acquired with eyes open and passive viewing of a cross hair (TR= 800ms,Resolution = 2.4 x 2.4 x 2.4 mm, TE = 30, 60 slices, FOV=216x216). T1-weighted,T2-weighted, and DWI images were also collected during this time (see Casey et al 2018 for further details).

Template Matching Method (see detailed explanation below image diagram)



Template matching method: 
A)  Whole-brain connectivity matrices were generated for each subject. The connectivity of each point in the brain is thresholded and then compared against network templates generated from an independent sample (thresholded to correlation values > Zscore=1 to preserve ~15.9% of the top connections). Here we show an example of connectivity of a grayordinate within the posterior cingulate cortex (PCC) to the default mode network (DMN), somatomotor dorsal network (SMD), and the salience network (SAL) (only 3 networks displayed for visualization purposes). We calculate an eta squared value for the comparison against each network and each grayordinate is then assigned to the network based on the maximum eta2 value. The grayordinate shown here has connectivity that resembles the default mode network.  
B) Each grayordinate was assigned a label for each subject. 
C) The probability of each network was calculated at each grayordinate. 
D) We thresholded the probability maps to produce a probabilistic ROI set of high network homogeneity using ABCD group1.

Each network was used to compare functional connectivity across ABCD group1 and group2. The list of networks included are the default mode network (DMN), the visual network (VIS), the frontoparietal network (FPN), the dorsal attention network (DAN), the ventral attention network (VAN), the salience network (Sal), the cingulo-opercular network (CO), the dorsal sensorimotor network (SMd), the lateral sensorimotor network (SMl), the auditory network(AUD), the temporal pole network (TP), the medial temporal network (MTL), the parietal occipital network (PON), and the parietal medial network (PMN).
